## 简介
一个类C语言的编译器。语言特性类似C语言，但是有更加方便的数学计算和嵌套函数。

## 文件结构
```
|─reports  报告
└─compiler 源码
  ├─tests   测试
  |  └─gtest 测试框架
  └─src     实现的源码
     ├─gen      目标代码生成
     ├─lexical  词法分析
     ├─parse    语法分析
     ├─semantic 语义分析
     └─util     辅助函数
```
